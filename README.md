# Social Networking App

# [Instagram Clone]

A completely Responsive Social App (A prominent try to clone instagram )- Works on Android, iOS & Web! 

## Features
- Responsive Instagram UI
- Email & Password Authentication
- Share Posts with Caption
- Display Posts with Caption
- Like & Comment on Posts
- Search Users
- Follow Users
- Display User Posts, Followers & Following
- EVERYTHING REAL TIME
- Sign Out



## Installation
After cloning this repository, migrate to ```social-App``` folder. Then, follow the following steps:
- Create Firebase Project
- Enable Authentication
- Make Firestore Rules
- Create Android, iOS & Web Apps
- Take Web FirebaseOptions and put it in main function in main.dart file replacing the keys 
Then run the following commands to run your app:
```bash
  flutter pub get
  open a simulator 
  flutter run
  flutter run -d chrome --web-renderer html (to see the best output)
```

## Tech Used
**Server**: Firebase Auth, Firebase Storage, Firebase Firestore

**Client**: Flutter, Provider
